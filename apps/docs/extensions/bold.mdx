## Keyboard Shortcuts

| Command | Shortcut | Description |
|---------|----------|-------------|
| toggleBold() | `⌘/Ctrl-b` | Toggle bold formatting |
| toggleBold() | `⌘/Ctrl-B` | Toggle bold formatting (uppercase) |
